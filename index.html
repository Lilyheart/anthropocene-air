Why this site exists

Human entry into the industrialization phase represented a major shift in our relationship with the Earth as an integrated system. Industrialization brought many positive advancements to humankind. For many, it increased health and wellbeing. Unfortunately, these changes also have had (and continues to have) many negative effects; both on humankind as well as the Earth System. Some of these negative effects can be seen in atmospheric readings.

This website visualizes the changes in different aerosols, many of them coming from purely anthropogenic sources.

History

  • 1970: Clean Air Act: Legislated the government's involvement in regulating air polution.
  • 1977: Clean Air Act Amendments: Stated that "air pollution prevention ... and air pollution control at its source is the primary responsibility of States and local governments"
  • 1985: IMPROVE project

How

What Data?

This site uses data available at the Federal Land Manager Environmental Database. I utilized data from both the IMPROVE aerosol and IMPROVE aerosol, RHR II datasets.

How is the data prepared?

The distribution of each parameter is analyzed prior to being added to the site. Percentiles are selected so that many of the data features are visible. These percentiles represent the minimum and maximum used for the color scale. Each value outside of this range is colored as if it were the minimum or maximum amount. The actual data value is available by hovering the mouse over the data point. The color scale represents at least 95% of the data of any parameter.
